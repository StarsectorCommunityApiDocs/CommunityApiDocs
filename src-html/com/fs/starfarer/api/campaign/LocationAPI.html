072         * @param focus
073         * @param category graphics category in settings.json

075         * @param bandWidthInTexture

078         * @param bandWidthInEngine

085                                         float bandWidthInEngine, float middleRadius, float orbitDays);

089         * Same as above, but with a "terrain" ring also being added.

092         * 
093         * Returns the terrain entity, NOT the RingBandAPI visuals.
094         * 
095         * @param focus
096         * @param category
097         * @param key
098         * @param bandWidthInTexture
099         * @param bandIndex
100         * @param color

105         * @param optionalName

142         * @param type id in custom_entities.json

155         * @param type id in custom_entities.json
